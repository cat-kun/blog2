开启PHP环境

  • PHP
随笔

说到PHP环境，大多数人会想到wampserver诸如此类的东西.今天要说的是在无论在哪个目录下都可以开启php环境，不用放到像wampserver的www目录底下。
例如：我现在有一个静态页项目，而里面需要测试接口，所以新建了个test.php之类的东西，但是如果为了测试一个接口就把整个目录迁移到www目录底下，会感觉比较麻烦，那么我们可以就在整个目录下开启php环境。具体怎么做呢？

首先安装php，从php官网下载，版本大家可以选择自己喜欢的下载哈！然后就是解压到某个目录下，这个随意，只要不是C盘就行。比如我是解压到E盘的目录下

当然还没完，需要在系统的环境配置下路径
“我的电脑”-右键“属性”-“高级系统设置”


在最后一个分号后写上刚才解压后的路径，比如我的是E:\Program Files\php-7.0.3\;
然后就是确定 确定了。怎么看是否配置成功了呢？

打开命令行窗口

windows系统是开始菜单 运行 输入 cmd，其他系统不造
然后在里面输入php -v 如果php的版本号，证明配置成功。

上面讲了一坨都是配置环境，具体怎么使用呢？好了，高潮来了
依然是在命令行窗口下，跳转到你项目所在的路径 cd 你的目录然后回车

接下来输入php -S localhost:8080 localhost是本机的意思 8080是端口号你喜欢的话 8888也行

看到这样的提示信息代表已经开启了php环境了，在浏览器输入http://localhost:8080就可以访问php页面了
