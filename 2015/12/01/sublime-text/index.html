Sublime Text的缩进问题

  • Sublime Text
随笔

很久之前就看到过说不要用一个tab去缩进代码，例如这样

可能有人会说，这样有啥问题呢？代码排版不是挺好的吗？在windows系统的sublime text看是挺好的，然而在浏览器的查看源码，就变成这样了

这根本就不是一个tab的宽度了呀~
那么，难道我们要按4个空格键去实现代码的缩进吗？答案是no

在sublime text中可以对它进行设置

菜单栏: Preferences -> Settings – More -> Syntax Specific – User

然后添加设置代码就可以了



{
    “tab_size”: 4,
    “translate_tabs_to_spaces”: true
}


然后按 ctrl+s 保存即可。


每一个tab缩进刚好是4个空格~

完美！！！！

其他编辑器或者IDE也是可以设置成一个tab等于4个空格
